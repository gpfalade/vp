How to run the VOTELINE site (a project site)

1. Download the  zip file

2. Extract the file and copy "vp" folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name "vote"

6. Import "vote.sql" file(given inside the zip package in database folder)

7. Run the script http://localhost/voteline (frontend)

8. For admin Panel http://localhost/vp/admin@admin_page
